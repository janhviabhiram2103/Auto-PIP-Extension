Privacy Policy for Auto PiP Video

Effective Date: 25-09-2025

1. Data Collection

Auto PiP Video does not collect, store, or transmit any personal information from users. The extension only enables Picture-in-Picture mode for videos on websites you visit.

2. User Data

This extension does not access, track, or log:

  • Personally identifiable information (name, email, address, etc.)
  • Health information
  • Financial or payment information
  • Authentication credentials or passwords
  • Personal communications (emails, texts, chat messages)
  • Location data (GPS, IP address, or region)
  • Web history or browsing activity
  • User interactions (clicks, scrolls, keystrokes)
  • Website content beyond video elements needed for PiP functionality

3. Sharing of Data

Auto PiP Video does not sell, share, or transfer any data to third parties.

4. Purpose of Permissions

Auto PiP Video does not require tabs or scripting permissions. It only interacts with video elements directly to enable Picture-in-Picture functionality.
